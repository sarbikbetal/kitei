## Online PDF compressor made with Node.js using bee-queue
Hosted on https://kitei.herokuapp.com/

### To run it locally
Rename `sample.env` to `.env` and fill in the details
then run
```sh
$ npm install gsx-pdf-optimize --global
$ npm install
$ npm start
```
